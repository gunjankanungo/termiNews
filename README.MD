Read the latest news headlines right in your terminal. Each headline is paired with a url to lead to the article.


## Getting Started

Install it via npm:

```shell
$ npm i -g terminews
$ news
```

### To Install Using GitHub
- ```$ git clone https://github.com/DanielDwyer/termiNews.git```
- ```$ cd termiNews```
- ```$ npm i --save```
- ```$ npm i -g .```
- ```$ news```
